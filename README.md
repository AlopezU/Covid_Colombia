# Coronavirus (Covid-19) Colombia. Repositorio de datos.  
Recursos para hacer análisis del Covid en Colombia, incluye datos oficiales y otros para hacer analítica de datos. 

La información oficial que reporta Colombia se encuentra en: https://www.ins.gov.co/Noticias/Paginas/Coronavirus.aspx Pero, debe descargarse cada noche para estar actualizado. Aquí estará la base de datos alojada.
